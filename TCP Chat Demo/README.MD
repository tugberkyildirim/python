# TCP Chat Demo

This project is a simple **TCP client-server application** written in Python.
Objective: To learn network programming with Python and send and receive messages.

> ⚠️ **Warning:** This application is configured to run only on **localhost**.  
> Running it on other computers or on the internet may pose a security risk.  

---
## Requirements
- Python 3.x

---

## Installation and Execution

1. Run the server:
```bash
python server.py
```

2. Run the client:
```bash
python init.py
```
