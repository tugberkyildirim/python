# TCP Chat Demo

Bu proje, Python ile yapılmış basit bir **TCP istemci-sunucu uygulaması**dır.  
Amaç: Python ile ağ programlamayı öğrenmek ve mesaj gönderip almak.

> ⚠️ **Uyarı:** Bu uygulama sadece **localhost** üzerinde çalışacak şekilde ayarlanmıştır.  
> Başka bilgisayarlarda veya internette çalıştırmak güvenlik riski oluşturabilir.  

---

## Gereksinimler
- Python 3.x

---

## Kurulum ve Çalıştırma

1. Sunucuyu çalıştırın:
```bash
python server.py
```

2. Client çalıştırın:
```bash
python init.py
```
